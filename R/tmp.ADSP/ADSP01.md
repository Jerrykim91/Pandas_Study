## 데이터의 종류 

### 정형 데이터

- 특징

	- 데이터 자체로 분석 가능
	- RDB구조의 데이터
	- 데이터베이스로 관리

- 유형

	- ERP
	- CRM
	- SCM 
	- 그외의 정보시스템...

### 반정형 데이터

- 특징

	- 데이터로 분석가능 하지만 해석이 불가능
	- 메타정보를 활용해야 해석 가능 

- 유형

	- 로그데이터
	- 모바일 데이터
	- 센싱데이터

### 비정형 데이터 

- 특징

	- 데이터 자체로 분석 불가능
	- 특정한 프로세스를 거쳐 분석데이터로 변경후 분석

- 유형

	- 파일 형태로 저장 , 관리
	- 영상,음성 문자등...
